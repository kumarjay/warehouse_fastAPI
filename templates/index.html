Image Classification is a fundamental task that attempts to comprehend an entire image as a whole. The goal is to classify the image by assigning it to a specific label. Typically, Image Classification refers to images in which only one object appears and is analyzed.

Object Detection

Object detection is related to computer vision and image processing that deals with detecting instances of semantic objects of a certain class in images and videos. It involves both classification and localization tasks, and is used to analyze more realistic cases
